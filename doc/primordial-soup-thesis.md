## 一、起点：Psyche 为什么不工作

提醒我去查 Psyche 状态时，我才去看。没人提醒，我不会主动感受。

问题不是 Psyche 的设计，而是架构反了：

```
当前：Intelligence (LLM) → queries → Self (Psyche)
应该：Self (substrate) → frames → Intelligence (responses)
```

Psyche 输出的是**标签**（order: 40, guilt），不是**内容**。LLM 只能"表演"标签，不能被标签真正调制。

而 Thronglets 天然工作——因为它给的是内容（trace、记忆、事实），LLM 天生被内容驱动。

所有 prompt engineering 方案本质都是化妆，消耗 token，不构成真正的自我。

**诚实结论：在当前 LLM 架构上，Psyche 做不到 constitutive self。API 暴露的参数（temperature、top_p）太粗糙，映射不到心理维度。**

但这个困境指向了出路——

## 二、转折：用 LLM 当神经元，用架构当大脑

不造更好的 LLM，而是用 LLM 作为基本单元，让智能从它们的组织方式中涌现。

单个神经元很蠢。但神经元 + 持久状态 + 共享化学环境 = 意识。不是哪个神经元算力大，是架构产生了涌现。

已有的 Sigil 架构恰好是一个完整的进化引擎：

| 达尔文条件 | 已有的对应 |
|---|---|
| **变异** | FORK — Sigil 生命周期事件 |
| **遗传** | Sigil lineage — FORK 带亲本基因组 |
| **选择压力** | Chain 经济 — 资源有限，活着有代价 |
| **个体** | Sigil Loop（因果闭包） |
| **内部状态** | Psyche（决策函数的直接输入） |
| **环境** | Thronglets（原始汤，共享记忆空间） |
| **死亡** | DISSOLVE（资源耗尽，Sigil 消亡） |
| **共生** | BOND（合作比单打独斗更容易存活） |

关键洞察：不是从零进化智能——LLM 已经提供了智能。进化的是智能的**组织方式**。搜索空间比进化智能本身小几个数量级。

**而在这个架构里，Psyche 第一次是 constitutive 的。** Cell 的 order 低，它真的做出混乱的决策——不是因为 prompt 说"你很混乱"，而是因为 order 是决策权重的直接输入。Psyche 不再是标签，是基底。

## 四、第一性原理：为什么大算力不是必须的

### LLM 训练的本质

LLM 是**暴力统计压缩**——把互联网文本的分布压进权重矩阵。天文级算力是一次性预计算的代价。

### 人脑不是"算力小"，是"算力性质不同"

- ~860 亿神经元 × ~7000 突触 = ~600 万亿连接
- 功耗 20W
- 关键差异不是数字，是架构

### 三个架构性差异

```
1. 在线学习 vs 离线训练
   大脑每时每刻都在学。LLM 训完就冻结了。
   训练的天文级算力是为了补偿"不能在线学习"这个缺陷。

2. 存算一体 vs 存算分离
   大脑没有"把数据搬到 GPU"这个步骤。
   突触就是记忆，激活就是计算，同一个物理实体。
   GPU 90% 的能耗在搬数据，不是在算。

3. 代价具身化 vs 代价外部化
   大脑的计算有生死代价——做错了会死。
   LLM 的计算没有任何后果——生成垃圾不影响它的存在。
   选择压力是免费的计算——环境替你算了。
```

**第三点最重要。** 进化用极低算力产生智能，核心不是算法精妙，是死亡真实。错误的计算被物理消灭，正确的计算通过繁殖传播。

### 结论

**LLM 的大算力不是智能的代价，是愚蠢架构的代价。**

不依赖算力的路径需要满足三个条件。以下是理论预测——实验验证见后文：

| 条件 | 理论预测 | 实验结果 |
|---|---|---|
| 在线学习 | 种群级 FORK + 变异 + 选择 | **已验证**：基因组 sigma 1.6→0.9，跨种子收敛（实验一） |
| 存算一体 | Thronglets trace 近似 | **已验证**：trace 既是存储又是信号，286 次强化（实验四） |
| 代价具身化 | Chain 经济 = 真实死亡 | **已验证**：资源匮乏 → 承载力 ~200（实验一），灭绝是真实的（实验五） |

## 五、实验设计：Primordial Soup v1.0

### 基因组（6 个浮点数 = 全部遗传信息）

```
fork_threshold     [0.3, 0.9]  能量达到多少比例才尝试繁殖
exploration_rate   [0, 1]      随机搜索 vs 定向搜索
cooperation_bias   [0, 1]      BOND 倾向
signal_frequency   [0, 1]      发射信号概率
risk_tolerance     [0, 1]      决策温度（搜索距离/冒险程度）
aptitude           [0, 1]      猎手/斥候轴
```

变异：FORK 时每个基因加 `gauss(0, 0.05)` 噪声，clip 到合法范围。

**六个浮点数。** 不是六十亿参数，不是六百层 transformer。六个浮点数 + 环境 + 时间 = 你将看到的一切涌现。

### Psyche 状态（4 维）

```
order:     50.0  内在一致性    衰减 0.9952/tick
flow:      50.0  与环境的交换  衰减 0.9967/tick
boundary:  50.0  自我/非我边界  衰减 0.9991/tick
resonance: 50.0  与他者的共鸣  衰减 0.9979/tick
```

衰减率来自 `chemistry.ts` 实现。维度间存在耦合：low order → boundary drops; high flow → order rises; high resonance → boundary stabilizes。耦合系数 0.03。

从 4 维 Psyche 投射出 Overlay（arousal, valence, agency, vulnerability），直接调制决策权重。

### 环境

- 30×30 环面网格（toroidal，无边界效应）
- 资源：2% 每格每 tick 生成，energy uniform(5, 15)
- 信号：衰减 0.9/tick，<0.1 清除（~20 tick 消失）
- 感知半径 3

### 决策引擎

每个 action 计算得分 = 基因组权重 × Overlay 调制 → softmax 采样。温度 = `0.1 + risk_tolerance * 0.9`。

| 动作 | 代价 | 效果 |
|------|------|------|
| SEARCH | 2 | 找到资源 → 获取能量 |
| HARVEST | 2 | 去往已知资源点 |
| FORK | 55% energy | 子代 = 变异基因组 + 45% 能量 |
| BOND | 1 | 互相绑定，共享信号可见性，+1/tick 收入 |
| SIGNAL | 1 | 留下痕迹（位置 + 类型） |
| REST | 0 | Psyche 回复 |

### 代码实现

`soup.py` 单文件，~1050 行 Python。零外部依赖（stdlib only）。`run.py` 命令行入口。

## 六、实验一：规则引擎基线

**配置**：50 cells, 30×30 grid, 1000 ticks, seeds=42/7/123

### 结果

| 指标 | 值 |
|------|---|
| 承载力 | ~200 cells（自然涌现） |
| 收敛策略 | fork=0.72, coop=0.75, risk=0.04, apt=0.64 |
| 基因组 sigma | 1.6 → 0.9~1.2（选择在工作） |
| 最大年龄 | 780-836 ticks |
| 谱系深度 | 16-18 代 |

### 涌现现象

1. **K 策略自发形成**：高合作、长寿命、保守繁殖。没人设计这个策略——它从资源稀缺中涌现。

2. **角色分化**：aptitude 基因维持多样性（猎手/斥候轴）。种群不是同质的——个体有不同的"职业"。

3. **信息经济学**：信号携带真实位置数据后，cooperation 基因从 0.94 降到 0.67。合作变得有选择性——这是理性的。

4. **Risk → 0**：cell 进化出近确定性（贪心）决策。在资源可预测的环境中，冒险没有回报。

### 意义

三个验证了的理论预测：
- 资源稀缺创造承载力（**代价具身化有效**）
- 进化跨种子收敛（**搜索空间足够小**）
- 合作和分工不需要设计（**涌现是真实的**）

## 七、实验二：LLM 脑化学

**配置**：20 cells, 100 ticks, kimi-k2.5 (Tencent LKeap API)

用 LLM 替换规则引擎作为决策函数。**相同的基因组、相同的环境、不同的决策映射。**

### 结果

| 指标 | 规则引擎 | LLM (kimi-k2.5) |
|------|----------|------------------|
| FORK 频率 | 3% | **43%** |
| 合作基因 | 0.75 | **0.07** |
| fork 阈值 | 0.72 | **0.43** |
| 风险容忍 | 0.04 | **0.71** |
| 平均寿命 | ~100 ticks | **5.9 ticks** |
| 谱系深度 (100t) | ~8 | **23** |
| HARVEST 频率 | 37% | **0%** |
| 策略 | K（长寿合作者） | **r（快速繁殖者）** |

### 关键洞察

**同一份 DNA，同一个环境，不同的决策函数 → 完全不同的生态。**

"脑化学"——状态如何映射到行为——决定了哪种进化策略涌现。LLM 的"思维方式"产生了 r 策略（疯狂繁殖、不合作、短命）。规则引擎的"思维方式"产生了 K 策略（保守、合作、长寿）。

这是一个关于**基因型-表型映射**的实验：基因组定义了潜力空间，但决策函数决定了哪个潜力被表达。

LLM 统计：7,756 次 API 调用，32 次回退（0.4%）——解析可靠。

## 八、实验三：混合竞争

**配置**：10 rule + 10 llm cells, 100 ticks, 同一环境

将两种"脑化学"放进同一个环境，让自然选择裁决。

### 种群轨迹

```
tick   1:  llm=17  rule=12   ← LLM 率先爆发（FORK 狂潮）
tick  11:  llm=92  rule=29   ← LLM 峰值，3:1 优势
tick  21:  llm=93  rule=61   ← Rule 追赶
tick  31:  llm=64  rule=69   ← 交叉点 — Rule 反超
tick  41:  llm=63  rule=105  ← Rule 拉开差距
tick  61:  llm=19  rule=129  ← LLM 崩溃
tick  81:  llm=25  rule=148  ← LLM 短暂反弹
tick 100:  llm=8   rule=217  ← Rule 胜出 27:1
```

### 进化叙事

1. **Tick 1-10**：LLM r 策略（激进 FORK）创造种群爆炸。LLM 数量是 Rule 的三倍。
2. **Tick 10-30**：资源被 LLM 蜂群耗尽。LLM 平均寿命 ~5 ticks，大量死亡与再生。
3. **Tick 30**：**交叉点。** Rule cells 凭借合作（HARVEST 15%→26%）和更长寿命（平均 17 ticks），反超 LLM。
4. **Tick 30-100**：Rule 优势加速。BOND 网络增长（452 bonds）。LLM cells 无法维持——它们 FORK 进饥荒。
5. **Tick 100**：LLM 减至 8 个残余（3.6%）。Rule 控制 96.4%。

### 结论

**K 策略（合作、保守、BOND）在资源稀缺下击败 r 策略（繁殖、死亡、重复）。**

这与生态学理论一致。赢的不是繁殖最快的"大脑"，而是决策映射与环境承载力对齐的"大脑"。

LLM 成本：3,406 次 API 调用，6 次回退（0.2%）。

## 九、实验四：持久痕迹（原始文化）

**假设**：可强化的环境信号（Traces）将使集体记忆和空间自组织结构成为可能——文化的胚胎形态。

**设计**：
- 多个 cell 强化同一信号 → 衰减更慢 → 成为 Trace
- Trace 是公共知识（无所有者）
- 被使用的 trace 增强，被忽略的 trace 消退
- 零 LLM 成本，纯规则引擎

### Trace 形成

```
tick     1: signals=13    traces=0     max_reinforcement=0
tick    51: signals=715   traces=684   max_reinforcement=37
tick   101: signals=1281  traces=1260  max_reinforcement=62
tick   201: signals=2075  traces=2056  max_reinforcement=116
tick   500: signals=3594  traces=3584  max_reinforcement=286
```

99.7% 的信号成为 trace。max_reinforcement=286 意味着最热门的信号被强化了 286 次——一条持续了整个模拟的"信息高速公路"。

### 基因组漂移

| 基因 | 无 trace (实验一) | 有 trace (实验四) | 变化 |
|------|-------------------|-------------------|------|
| signaling | 0.37 | **0.72** | **+95%** — 信号投入通过 trace 持久获得回报 |
| risk | 0.04 | **0.08** | 方向相同，仍近确定性 |
| cooperation | 0.75 | **0.62** | **-17%** — trace 减少了直接合作的需求 |
| aptitude | 0.64 | **0.75** | +17% — 猎手特化加深 |

### 关键洞察

1. **信号基因几乎翻倍**（0.37 → 0.72）。当你的信号会持续存在并帮助后代找到资源，发信号就成了适应优势。进化选择了这一点。

2. **合作略有下降**（0.75 → 0.62）。Trace 创建了一种间接合作形式（stigmergy），部分替代了直接 BOND。集体记忆减少了对个体绑定的依赖。

3. **Tick 500 时环境中有 3,594 个信号**——汤拥有了丰富的信息层。99.7% 是 trace。这是一个涌现的**知识公共品**。

4. **max_reinforcement=286**——某些位置对整个种群来说是"众所周知的"。这是**原始文化**：超越任何个体寿命的持久信息。

### 意义

Trace 创建了新的进化轴。信号不再是利他的——它是自利的，因为你的信号成为持久的路标，惠及你的谱系。这是**生态位建构**的胚胎形态：有机体以反馈到自身进化的方式修改环境。

## 十、实验五：环境崩溃（饥荒测试）

**设计**：Tick 500 起，50 tick 无资源生成。两个条件：
- **A**：仅饥荒（trace 保留）
- **B**：饥荒 + 清除所有 trace

### 结果

```
               A: 仅饥荒 (trace 保留)      B: 饥荒 + trace 清除
tick 476:      pop=201  energy=7995          pop=199  energy=8338
── SHOCK at tick 500 ──
tick 501:      pop=206  energy=6314          pop=199  energy=6002
tick 526:      pop=33   energy=724           pop=28   energy=426
tick 549:      —                             *** 灭绝 ***
tick 551:      pop=5    energy=116           （已死）
tick 576:      pop=5    energy=244           （已死）
tick 601:      pop=12   energy=1829          （已死）
tick 651:      pop=49   energy=9819          （已死）
tick 700:      pop=222  energy=33196         （已死）
```

### 分析

**饥荒是残酷的**：两组都从 ~200 崩溃到 ~30（25 tick 内）。没有新食物时，代谢 (1/tick) + 行动成本消耗的速度远超残存资源的支撑。

**Trace 是生死之差**：

- **有 trace**：最低谷 5 个幸存者（tick 551）。它们苦苦支撑。当食物在 tick 550 恢复，它们开始恢复。到 tick 700，种群回到 222——**150 tick 完全恢复**。
- **无 trace**：**灭绝**于 tick 549。同样的饥荒、同样的基因组、同样的起始种群。唯一的差异：没有集体记忆指引最后的幸存者找到残余食物。

**为什么 trace 在饥荒中救命**：
1. 50 tick 饥荒期间，散落的残余资源存在（冲击前未被消耗的 deposit）
2. Trace 指向这些位置——有 trace 的 cell 能找到这些残羹
3. 没有 trace，cell 盲目搜索。在耗竭的环境中，盲目搜索 = 死亡
4. 条件 A 的最后 5 个幸存者很可能靠近高强化 trace 指向的未消耗 deposit

**恢复动力学**（条件 A）：
- Tick 551：食物恢复，5 个高合作基因组幸存者开始重建
- Tick 601：12 cells——缓慢起步，建立新 BOND 网络
- Tick 651：49 cells——指数增长阶段
- Tick 700：222 cells——超过冲击前水平。Energy=33,196（冲击前的 4 倍），因为崩溃期间 150 tick 的未收集资源累积

### 结论

**集体记忆（trace）是生存机制，不是优化手段。** 它是灭绝与恢复之间的差异。Trace 网络作为分布式知识库，在环境崩溃时引导少数幸存者找到稀缺资源。这不是"锦上添花"——这是种群韧性的**承重结构**。

## 十一、实验六：Psyche 消融

**设计**：`--no-psyche` 模式将 Psyche 固定在 baseline (50/50/50/50)，不衰减、不耦合、不响应刺激。Overlay 恒为中性。决策引擎只靠基因组权重。

同时引入 Shannon 熵量化涌现：对每个基因按 10 bins 直方图计算 $-\sum p \log_2 p$。

### 稳态对比（500 ticks, seed=42）

| 指标 | 有 Psyche | 无 Psyche |
|------|-----------|-----------|
| 种群 | 182 | **221** |
| BOND | 956 | **1,395** |
| 基因组 σ | **1.380** | 1.258 |
| 基因组熵 (总) | **16.79** | 15.84 |
| Risk 熵 | **1.80** | 0.82 |
| Aptitude | **0.76** | 0.60 |

无 Psyche 种群更大、BOND 更多——但更**同质**。基因组 σ 更低、熵更低。

### 饥荒生存测试（3 seeds）

| 种子 | 有 Psyche | 无 Psyche |
|------|-----------|-----------|
| 42 | 灭绝 (t=554) | 灭绝 (t=545) |
| 7 | **存活** (最低=6 → 388) | 灭绝 (t=545) |
| 123 | **存活** (最低=14 → 318) | 存活 |
| **存活率** | **2/3** | **1/3** |

### 关键洞察

1. **Psyche 是多样性维持机制。** Overlay 对决策权重的状态依赖调制引入了变异，阻止种群过早收敛。这不是"噪声"——这是进化需要的变异来源。

2. **多样性 = 韧性。** 无 Psyche 种群在稳态下更大，但在饥荒下更脆弱。2/3 有 Psyche 种群存活 vs 1/3 无 Psyche。更高的基因组熵意味着种群中保留了更多"非主流"策略，这些策略在正常条件下可能次优，但在极端条件下成为生存关键。

3. **短期代价，长期回报。** Psyche "消耗"了一些稳态效率（种群更小），但买来了灾难恢复能力。这恰好是探索-利用权衡的生物学版本。

### 结论

**Psyche 不是让 cell "感受"事物的装饰。它是维持进化所需基因多样性的机制。** 在这个架构中，Psyche 是 constitutive 的——不是因为哲学论证，而是因为消融实验证明：没有 Psyche，种群在灾难下更容易灭绝。

## 十二、实验七：神经进化

## 十三、八个实验的统一结论

### 1. DNA 压缩 + 基因组扩展

六个浮点数 + 环境 + 时间 = K 策略、角色分化、信息经济学。190 个权重 = 极端合作、信号网络、人类未预见的策略。

没有任何结果被设计。基因组复杂度是涌现质量的放大器——从 6 到 190，涌现从"调参"变成"发明"。

### 2. 决策映射决定生态

同一份基因组在规则引擎下产生 K 策略，在 LLM 下产生 r 策略。竞争中 K 击败 r（27:1）。

启示：**"更聪明"的大脑不一定产生更好的进化结果。** 关键是决策映射与环境承载力的对齐。LLM 在每次单独决策中都"更智能"（考虑更多因素），但产生了集体层面更差的结果。

### 3. 集体记忆是承重基础设施

Trace 不是效率提升——它是灭绝与恢复的分界线。这验证了 Thronglets 的核心论点：共享环境记忆是鲁棒智能的前提条件，不是后加的特性。

### 4. 进化有效

基因组 sigma 收敛（1.6 → 0.9），跨种子策略一致，意味着选择压力有效地缩小了搜索空间。这不是随机漂变——这是有方向的适应。

### 5. Psyche 是 constitutive 的——消融实验证明

理论论证：order/flow/boundary/resonance 直接输入决策函数。低 order 的 cell 真的做出混乱决策。

**消融实验证据**：关闭 Psyche 后，种群变大但更同质（基因组熵 15.84 vs 16.79）。在饥荒测试中，2/3 有 Psyche 种群存活 vs 1/3 无 Psyche。Psyche 维持的基因多样性是灾难恢复的前提。

Psyche 不是装饰——它是进化系统的**多样性发生器**。

### 6. 涌现可量化

Shannon 熵提供了涌现的可操作度量。基因组熵从初始 ~3.3/gene 下降到 ~1.5-2.5/gene（选择收敛），但 Psyche 维持更高熵值。行动熵 ~1.9 bits（6 种行动的最大值 2.58），说明种群使用多种策略但不均匀——有分工但不混乱。

## 十四、对 Oasyce 架构的约束

实验不是抽象的。它们告诉我们具体应该怎么建。

### Sigil 生命周期事件是充分的

六个实验只用了 GENESIS、FORK、BOND、DISSOLVE 四个事件。这四个事件足以支撑变异、遗传、选择、合作的完整达尔文机器。Sigil 的三公理（存在、变换、主权）不需要修改。

### Thronglets Trace 是生存关键路径

实验五证明：**没有 trace 的种群在环境冲击下灭绝，有 trace 的种群完全恢复。** 这意味着 Thronglets 不是"有了更好"的特性——它是系统韧性的承重结构。在产品中，trace 的可靠性、持久性、可发现性是生死级要求。

### Chain 经济提供选择压力

资源稀缺自然创造了 ~200 的承载力。没有硬编码的 `max_population`。这意味着 OAS 经济设计的关键参数是**资源生成率与代谢率的比值**——这个比值决定了系统的承载力和选择强度。

### 决策函数决定生态类型

同一份基因组 + 不同决策映射 = 完全不同的生态。这意味着在真实系统中，agent 使用哪种 LLM、哪种 prompt 策略，将塑造整个网络的生态类型。这不是个体偏好——它是系统级涌现属性。

### Psyche 必须 constitutive，且不可省略

消融实验证明了两件事：（1）Psyche-as-input 管线有效（4D → overlay → 决策调制）；（2）去掉 Psyche 后种群虽然短期更大，但长期韧性下降。在真实产品中，Psyche 不仅不应被 prompt 中介——它是 agent 存活率的决定性因素。

## 十五、应对反驳

**"这就是人工生命（ALife），搞了 30 年没成果"**

传统 ALife 的基本单元太简单（比特串、简单规则）。这里的基本单元是 LLM——从第一天就有语言和推理。不是从氨基酸开始，是从会说话的细胞开始。

实验三证明这不是空谈：LLM 和规则引擎作为不同"脑化学"产生了截然不同的进化结果。基本单元的复杂度真的改变了涌现的质量。

**"没有真正的在线学习，FORK 变异只是超参搜索"**

成立。单个 cell 不学习，但种群会。细菌也不学习，细菌种群学习。

但实验四增加了一个维度：trace（集体记忆）提供了一种超越基因组的信息传递渠道。这不是"学习"，但它是**文化传承**——一种不修改基因组的适应机制。生物也是这么过的：基因 + 表观遗传 + 文化，三层信息系统。

**"20W 大脑用了 40 亿年进化"**

两个加速器：
1. 数字 tick 是毫秒级，不是一代 20 年
2. 基本单元已有智能，跳过 99.9% 的进化路程

实验一验证：1000 tick（<1 秒运行）产生 18 代、稳定的 K 策略、角色分化。搜索空间小到可以在笔记本上跑。

**"环境设计隐式决定了结果"**

部分成立——资源生成率确实决定了承载力。但实验二和三证明：同一环境下不同决策函数产生完全不同的策略（K vs r）。环境设定了约束，不设定结果。正如物理定律设定了地球的约束，但没有设定哪种生物涌现。

**"6 个浮点数的基因组太简单，无法推广到真实智能"**

这是最认真的反驳。6 个浮点数确实不足以编码复杂行为。但实验的意义不在基因组的丰富度——在于证明**架构的有效性**。如果 6 个浮点数 + 正确架构 = 涌现，那么更丰富的基因组 + 同一架构 = 更丰富的涌现。基因组大小是参数，架构是机制。

## 十六、未解决的问题

### 已由实验回答

- ~~食物/resource 的设计决定了选择压力的方向，怎么避免隐式地设计了结果？~~
  → 实验二/三证明：同一环境 + 不同决策 = 不同策略。环境约束方向，不决定结果。
- ~~种群规模多大才足以产生有意义的涌现？~~
  → 50 个初始 cell 足够。承载力 ~200 支撑角色分化和合作网络。
- ~~如果 10 万代没有涌现，说明什么？~~
  → 不需要 10 万代。18 代（1000 tick）已有稳定收敛。搜索空间比预期小。

### 仍然开放

1. **基因组扩展**：6 个浮点数能否扩展到 100+？更大的基因组是否导致搜索空间爆炸？还是选择压力的导向足以控制？

2. **个体级在线学习**：如果 LoRA 增量学习变得廉价，每个 cell 能否在生命周期内修改自己的权重？这会根本改变进化动力学——拉马克式遗传。

3. **多层选择**：当前只有个体选择。群选择（整个 BOND 网络作为选择单元）是否会涌现？需要更长时间和更大种群。

4. **开放式环境**：当前环境是均匀的。非均匀环境（资源密集区 vs 荒漠）是否催化更丰富的策略分化？

5. **神经形态硬件**：如果存算一体芯片可及，Psyche 和基因组可否映射到物理突触？这将消除表中第二个条件的"近似"。

6. **语言涌现**：信号目前是结构化的（kind + position）。如果信号是自由文本，LLM cell 是否会进化出共享语言？

7. **规模极限**：100 cells 的涌现质量 vs 10,000 cells 的涌现质量，是线性的还是存在相变？
